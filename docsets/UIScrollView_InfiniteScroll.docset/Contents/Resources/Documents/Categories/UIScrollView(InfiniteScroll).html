  • infiniteScrollTriggerOffset

    Set vertical adjustment for scroll coordinate used to determine when to call handler block. Non-zero value advances the point when handler block is being called making it fire by N points earlier before scroll view reaches the bottom. This value is measured in points and must be positive number. Default: 0.0
